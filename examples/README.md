# Sample Trails · 示例轨迹

**🌐 [中文](README.md) · [English](README.en.md)**

这里提供一组「**格聂牧场+V**」四天穿越路线的 KML 样本，来自 [两步路](https://www.2bulu.com) 三位不同用户上传的变体。

## 用途

- 直接体验多轨迹叠加（同一路线不同人走的差异）
- 测试主轨迹切换、标注点对比、分组、批量迁移、导出等所有功能
- 你可以基于这些样本 fork 出自己的路线库

## 使用方法

1. 打开 `hiking-trail-mapper.html`
2. 点 `➕ 添加轨迹`
3. **多选**本目录下所有 `.kml` 文件
4. 一次导入 3 条同区域轨迹

或者更简单：把这 3 个文件一起拖入浏览器窗口。

## 文件说明

| 文件 | 大小 | 备注 |
|------|------|------|
| `格聂牧场+V.kml` | ~849KB | 主线路（默认主轨迹）|
| `2025-07-1608_48格聂牧场+V.kml` | ~874KB | 2025-07-16 上传的变体 |
| `格聂牧场+v线.kml` | ~239KB | 简化版本，标注点较少 |

## 路线概况

- **地区**：四川康定西南 · 格聂神山周边
- **天数**：4 天
- **总里程**：~60 km
- **累计爬升**：~2500 m（阈值累加器算法）
- **最高海拔**：~4845 m（垭口）
- **难度**：中等偏难，高海拔徒步
