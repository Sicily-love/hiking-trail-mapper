# Sample Trails · Example trails

**🌐 [中文](README.md) · [English](README.en.md)**

This directory contains a set of KML samples from three different [2bulu](https://www.2bulu.com) users hiking the same 4-day **"Genie Pasture+V"** trail.

## Purpose

- Experience multi-trail overlay firsthand (see how different hikers took slightly different paths on the same route)
- Test primary trail switching, waypoint comparison, grouping, batch move, and export features
- Fork these as a starting point for your own trail library

## How to use

1. Open `hiking-trail-mapper.html`
2. Click `➕ Add Trail`
3. **Multi-select** all `.kml` files in this directory
4. Import three trails covering the same region at once

Or simpler: drag all three files into the browser window at once.

## Route summary

- **Region**: Sichuan Kangding, southwest of Mt. Genie
- **Days**: 4
- **Distance**: ~60 km total
- **Ascent**: ~2500 m (via threshold accumulator algorithm)
- **Max altitude**: ~4845 m (pass)
- **Difficulty**: moderate-hard, high-altitude trekking
