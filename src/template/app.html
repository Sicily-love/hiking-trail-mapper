📖 怎么从两步路导出 KML
1. 打开两步路轨迹页（如 2bulu.com/track/t-xxx.htm）
2. 找右上角或左侧工具栏的「导出」按钮
3. 选择 KML 格式下载到本地
4. 拖到这里

其他平台：六只脚 / Strava / Gaia GPS / All Trails / Garmin Connect 都有「Export KML」选项。
